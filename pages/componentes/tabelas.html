Tabela simples

Use table para organizar dados em linhas e colunas com cabeçalhos associados.

Exemplo

Cadastro de pessoas
Nome Cargo Cidade
Ana Teste São Paulo
Bruno Produto Curitiba

Código gerado

<table>
  <caption>Cadastro de pessoas</caption>
  <thead>
    <tr>
      <th scope="col">Nome</th>
      <th scope="col">Cargo</th>
      <th scope="col">Cidade</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">Ana</th>
      <td>Teste</td>
      <td>São Paulo</td>
    </tr>
  </tbody>
</table>

Explicação

  • Tag th: cria a célula de cabeçalho da tabela.
  • Atributo scope: define a associação do cabeçalho, com scope="col" para coluna e scope="row" para linha.
  • Tag thead: agrupa a área de cabeçalhos da tabela.
  • Tag tr: cria cada linha da tabela.
  • Tag tbody: agrupa as linhas de dados da tabela.
  • Tag td: cria células de dados comuns.
  • Tag caption: apresenta o título da tabela.

Tabela mesclada

Use colspan e rowspan para mesclar colunas e linhas quando a informação ocupar mais de uma célula.

Exemplo

Agenda semanal
Horário Segunda Terça Quarta
08h Reunião Planejamento Leitura
09h Treinamento

Código gerado

<table>
  <caption>Agenda semanal</caption>
  <thead>
    <tr>
      <th scope="col">Horário</th>
      <th scope="col">Segunda</th>
      <th scope="col">Terça</th>
      <th scope="col">Quarta</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">08h</th>
      <td rowspan="2">Reunião</td>
      <td>Planejamento</td>
      <td>Leitura</td>
    </tr>
    <tr>
      <th scope="row">09h</th>
      <td colspan="2">Treinamento</td>
    </tr>
  </tbody>
</table>

Explicação

  • rowspan="2": faz a célula ocupar duas linhas, da linha atual até a próxima.
  • colspan="2": faz a célula ocupar duas colunas, da coluna atual até a próxima.
  • Essa técnica deve ser usada com cuidado para não comprometer a leitura da tabela.

Tabela com checkbox

Use checkbox dentro de tabelas quando for necessário selecionar mais de um item ao mesmo tempo.

Exemplo

Tarefas do dia
Selecionar tarefa Tarefa Responsável
Revisar acessibilidade Gabriella
Validar contraste Ana

Código gerado

<table>
  <caption>Tarefas do dia</caption>
  <thead>
    <tr>
            <th scope="col">Selecionar tarefa</th>
      <th scope="col">Tarefa</th>
      <th scope="col">Responsável</th>
    </tr>
  </thead>
  <tbody>
    <tr>
            <th scope="row"><input type="checkbox" id="chk-1" aria-label="Tarefa Revisar acessibilidade"><label for="chk-1"></label></th>
      <td>Revisar acessibilidade</td>
      <td>Gabriella</td>
    </tr>
        <tr>
            <th scope="row"><input type="checkbox" id="chk-2" aria-label="Tarefa Validar contraste"><label for="chk-2"></label></th>
            <td>Validar contraste</td>
            <td>Ana</td>
        </tr>
  </tbody>
</table>

Explicação

  • checkbox: permite selecionar vários itens ao mesmo tempo.
  • label: amplia a área clicável e associa o texto ao controle.
  • th scope="row": ajuda a associar a seleção à linha correspondente.

Tabela com radio button

Use radio button quando apenas uma opção pode ser escolhida em um grupo.

Exemplo

Forma de contato preferida
Escolher canal Canal Descrição
E-mail Resposta por mensagem eletrônica
Telefone Contato por ligação

Código gerado

<table>
  <caption>Forma de contato preferida</caption>
  <thead>
    <tr>
            <th scope="col">Escolher canal</th>
      <th scope="col">Canal</th>
      <th scope="col">Descrição</th>
    </tr>
  </thead>
  <tbody>
    <tr>
            <th scope="row"><input type="radio" name="contato" id="radio-1" aria-label="Canal E-mail"><label for="radio-1"></label></th>
      <td>E-mail</td>
      <td>Resposta por mensagem eletrônica</td>
    </tr>
        <tr>
            <th scope="row"><input type="radio" name="contato" id="radio-2" aria-label="Canal Telefone"><label for="radio-2"></label></th>
            <td>Telefone</td>
            <td>Contato por ligação</td>
        </tr>
  </tbody>
</table>

Explicação

  • radio: permite escolher apenas uma opção dentro do mesmo grupo.
  • name igual nos radios: garante que apenas uma opção seja selecionada.
  • label: facilita a seleção e melhora a usabilidade.

Tabela com link e botão

Use links para navegar e botões para executar ações diretamente em cada linha.

Exemplo

Atendimentos
Protocolo Status Ação
12345 Aberto Ver detalhes

Código gerado

<table>
  <caption>Atendimentos</caption>
  <thead>
    <tr>
      <th scope="col">Protocolo</th>
      <th scope="col">Status</th>
      <th scope="col">Ação</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th scope="row">12345</th>
      <td>Aberto</td>
      <td>
        <a href="../contato.html" aria-label="Ver detalhes do protocolo 12345">Ver detalhes</a>
                <button type="button" aria-label="Fechar atendimento do protocolo 12345">Fechar</button>
      </td>
    </tr>
  </tbody>
</table>

Explicação

  • a: navega para outra página ou recurso.
  • button: executa uma ação sem mudar de página.
  • Os dois elementos podem coexistir na mesma célula quando a interface exige navegação e ação.

Tabela com ordenação

Os cabeçalhos são botões que alternam entre ordem crescente, decrescente e padrão, mantendo o foco no próprio botão e informando o resultado da ordenação.

Exemplo

Ranking de prioridade
Bruno 2 Curitiba
Ana 1 São Paulo
Carla 3 Recife

Código gerado

<table id="tabela-ordenavel">
  <caption>Ranking de prioridade</caption>
  <thead>
    <tr>
    <th scope="col"><button type="button" class="botao-ordenar" data-key="nome" aria-label="Ordenar por Nome, ordem padrão">Nome</button></th>
    <th scope="col"><button type="button" class="botao-ordenar" data-key="prioridade" aria-label="Ordenar por Prioridade, ordem padrão">Prioridade</button></th>
    <th scope="col"><button type="button" class="botao-ordenar" data-key="cidade" aria-label="Ordenar por Cidade, ordem padrão">Cidade</button></th>
    </tr>
  </thead>
  <tbody>
    <tr data-nome="Bruno" data-prioridade="2" data-cidade="Curitiba">
      <th scope="row">Bruno</th>
      <td>2</td>
      <td>Curitiba</td>
    </tr>
  </tbody>
</table>

Explicação

  • th com botão: permite ordenar a coluna sem perder a semântica de cabeçalho.
  • aria-label: informa a ação e o estado atual do botão, como ordem padrão, crescente ou decrescente.
  • O foco permanece no botão porque a tabela é reordenada sem recriar o elemento clicado.
  • O feedback confirma que a ordenação foi concluída.
