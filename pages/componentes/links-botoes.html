Link interno

Use links internos para mover a pessoa pela própria página ou para outra parte do site.

Exemplo

Voltar para o índice

Ir para Botões

Código gerado

<a href="#indice" aria-label="Voltar para o índice da página">Voltar para o índice</a>

<a href="#botoes">Ir para Botões</a>

Explicação

  • a: cria um link navegável.
  • href: aponta para uma âncora interna dentro da página ou do site.
  • aria-label: ajuda tecnologias assistivas a entender o destino do link com mais clareza quando esse precisa de um complemento.
  • O texto visível do link já é o rótulo nativo em HTML, então ele costuma ser suficiente sozinho quando o destino é claro.

Link externo

Use links externos quando o destino estiver em outro site. Sempre avalie target e rel.

Exemplo

Acessar WAI do W3C

Código gerado

<a href="https://www.w3.org/WAI/"
   target="_blank"
   rel="noopener noreferrer"
   aria-label="WAI W3C, abre em nova aba">
   Acessar WAI do W3C
</a>

Explicação

  • a: cria o link externo.
  • href: define o endereço completo do destino.
  • target="_blank": abre em nova aba.
  • rel="noopener noreferrer": evita que a nova aba acesse a página de origem e reforça privacidade.
  • aria-label: complementa o texto visível com uma informação extra, como avisar que o link abre em nova aba. O texto visível continua sendo o rótulo nativo do link, e o aria-label passa a ser o nome acessível lido por tecnologias assistivas.
  • No exemplo, aparecem dois rótulos porque um é o texto visível para todas as pessoas e o outro é o nome acessível usado pela tecnologia assistiva.

Botões

Use button ou input para ações de interface, definindo o tipo correto para cada comportamento.

Exemplo

Código gerado

<button type="button">Botão comum</button>
<button type="submit">Enviar</button>
<button type="reset">Limpar</button>

<input type="button" value="Input button">
<input type="submit" value="Enviar">
<input type="reset" value="Limpar">

Explicação

  • button type="button": botão comum para ações com JavaScript, sem enviar formulário.
  • button type="submit": envia o formulário associado.
  • button type="reset": limpa os campos do formulário para o estado inicial.
  • input type="button": alternativa ao button para ações simples. O texto mostrado vem do value, que é o rótulo visível nativo do input.
  • input type="submit": envia os dados do formulário. O value define o texto visível, como Enviar.
  • input type="reset": restaura os campos ao valor inicial. O value define o texto visível, como Limpar.
  • Assim como nos links, o texto visível do button e o value do input já funcionam como rótulo nativo em HTML.
