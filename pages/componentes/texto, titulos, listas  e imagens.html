Texto

Use a tag p para criar parágrafos de conteúdo textual.
Use br para quebrar uma linha dentro do mesmo parágrafo quando necessário.

Exemplo

Este é um exemplo de parágrafo criado com a tag p.
Esta linha foi quebrada com a tag br.

Código gerado

<p>Este é um exemplo de parágrafo criado com a tag p.<br>Esta linha foi quebrada com a tag br.</p>

Explicação

  • p: define um parágrafo semântico de texto.
  • br: faz quebra de linha no conteúdo textual sem iniciar um novo parágrafo.
  • O conteúdo entre abertura e fechamento da tag é o texto exibido na tela.

Títulos

Use h1 até h6 para organizar a hierarquia do conteúdo, do geral ao mais específico, h1 é o título principal.

Exemplo

Título nível h1
Subtítulo nível h1

Título nível h2
Complemento nível h2

Título nível h3

Título nível h4

Título nível h5
Título nível h6

Código gerado

<h1>Título nível h1<br>Subtítulo nível h1</h1>
<h2>Título nível h2<br>Complemento nível h2</h2>
<h3>Título nível h3</h3>
<h4>Título nível h4</h4>
<h5>Título nível h5</h5>
<h6>Título nível h6</h6>

Explicação

  • h1: representa o título principal da página.
  • h2 a h6: subtítulos em ordem hierárquica, sem pular níveis.

Listas

Existem três tipos principais de lista: não ordenada, ordenada e de definição.

Exemplo

Lista não ordenada (ul): itens sem ordem numérica.

  • HTML semântico
    Estrutura acessível
  • Acessibilidade
  • Testes manuais

Lista ordenada (ol): itens em sequência numerada.

  1. Planejar conteúdo
  2. Escrever código
    Revisar semântica
  3. Validar acessibilidade

Lista de definição (dl): termo e descrição.

WCAG
Conjunto de diretrizes internacionais de acessibilidade web.
ARIA
Atributos para complementar semântica em componentes interativos.
Devem ser usados com critério.

Código gerado

<ul>
    <li>HTML semântico<br>Estrutura acessível</li>
  <li>Acessibilidade</li>
  <li>Testes manuais</li>
</ul>

<ol>
  <li>Planejar conteúdo</li>
    <li>Escrever código<br>Revisar semântica</li>
  <li>Validar acessibilidade</li>
</ol>

<dl>
  <dt>WCAG</dt>
  <dd>Conjunto de diretrizes internacionais de acessibilidade web.</dd>
  <dt>ARIA</dt>
    <dd>Atributos para complementar semântica em componentes interativos.<br>Devem ser usados com critério.</dd>
</dl>

Explicação

  • ul: lista sem ordem de prioridade entre itens.
  • ol: lista com sequência explícita de passos ou ranking.
  • li: cada item individual da lista ul ou ol.
  • dl: agrupa termos e definições.
  • dt: termo principal a ser definido.
  • dd: descrição do termo definido.

Imagens

Use img com alt para descrever imagens informativas. Para imagem decorativa, use alt vazio.

Exemplo


Código gerado

<img src="https://via.placeholder.com/320x180/1d4ed8/ffffff?text=Componente+A11Y"
     alt="Banner azul com o texto Componente A11Y">

    <br>

<img src="https://via.placeholder.com/160x90/93c5fd/0f172a?text=Decorativa"
     alt="">

Explicação

  • img: insere uma imagem na página.
  • alt: descreve imagens informativas para tecnologias assistivas.
  • br: força uma quebra de linha quando você precisa separar blocos visuais no conteúdo.
